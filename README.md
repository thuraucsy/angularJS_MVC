# Installation
1. cd /var/www/html (for linux) or cd /c/xampp/htdocs (for windows) or cd ~/Sites/ (for mac)
2. git clone https://github.com/thuraucsy/angularJS_MVC.git 
3. cd angularJS_MVC
4. run it http://localhost/angularJS_MVC/guidebook.html#/
